# MatView App Cursor

## 📱 프로젝트 소개
맛집 리뷰와 노트 기능을 결합한 하이브리드 앱으로, 사용자들이 개인 노트처럼 맛집 리뷰를 작성하고 선택적으로 커뮤니티와 공유할 수 있는 플랫폼입니다.

## 🚀 주요 기능
### 기본 노트 기능
- [ ] 메모 작성 (최대 1만자)
- [ ] 카테고리별 분류
- [ ] 색상 태그 지정
- [ ] 검색 기능 (제목/본문)
- [ ] 사진 첨부
- [ ] 폴더 구조화
